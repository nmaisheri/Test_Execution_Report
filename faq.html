Frequently Asked Questions

What is this automation framework?
This is a real-time pytest automation dashboard that provides comprehensive testing capabilities with live monitoring and reporting. The framework supports marker-based test execution, allowing you to run specific test groups (smoke, regression, UI, performance, security) with real-time status updates and detailed error reporting. It features an interactive HTML dashboard with auto-refresh capabilities, making it ideal for continuous integration and test monitoring workflows.
How do I interpret the test results?
Test results are color-coded for easy identification: Green indicates passed tests, Red shows failed tests, Orange represents error states, Gray shows skipped tests, and Yellow indicates tests currently in progress. Each test displays execution time, markers, and detailed exception logs for failures. The summary section provides overall statistics including pass/fail rates and total execution time.
What do the different status colors mean?
The status colors provide quick visual feedback: Green (Passed) - Test executed successfully, Red (Failed) - Test failed with assertion errors, Orange (Error) - Test encountered unexpected errors, Gray (Skipped/Not Selected) - Test was skipped or not selected for execution, Yellow (In Progress) - Test is currently running.
